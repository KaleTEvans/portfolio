Stock Dashboard

View Application

This is an ongoing side project I have been working on. Right now I have been pulling data from APIs such as the Yahoo Data API, AlphaVantage, and Stock News APIs. I believe the stock market lately has been heavily reactive to news and general sentiment, and I am trying to build a platform that can follow trends and sentiment and combine a bit of technical analysis to produce stock recommendations. I also plan on adding my own API to save the recommended stocks and follow their price movement in order to see which analysis methods work the best and iterate on my methods to produce more accurate predictions. The project's GitHub can be found at https://github.com/KaleTEvans/stocksV2

Reddit Stock Tracker

View GitHub

This is another ongoing side project I have been working on. The purpose of this program is to pull data from the reddit API, specifically from the WallStBets forum, and search for traffic on different stocks. I started working on this because I wanted to experiment with creating a search engine, with the future possibility of creating my own APIs for the above stock project that have been personalized to give me the data I specifically want. As of right now, the app is very limited, and only scans the title of the hot posts on the forum for ticker symbols, and uses a node sentiment analyzer package (that does not understand reddit lingo very well) to rate the sentiment of the title, and uploads this data to an SQL database where I am able to manipulate and visualize the outcomes. I plan to add several upgrades in the future to make a cleaner and broader search and sentiment scorer, but getting the foundations has allowed me to learn a lot about how sentiment trackers like the Stock News API I have used in my stock project work.
